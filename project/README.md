# Project Memory

This directory contains internal project notes, plans, decisions, and open questions.

It is not user-facing documentation. Public documentation belongs in `docs/` if and when a documentation site is added.

## Structure

- `notes/` contains discoveries, requirements, research, and open questions.
- `plans/` contains implementation plans and PRP workflow artifacts.
- `decisions/` contains durable architecture and tooling decisions.

## PRP Workflow

Use `project/plans/` as the local PRP and implementation-plan directory. Do not create a top-level `plans/` directory for this repository.
